Dashboard

The Athlyte Search Dashboard is a powerful tool that allows you to search for sports data using natural language searches. It supports various sports and provides detailed statistics on players and teams.

Click here to know more about how to write natural language searches.

Sports

Currently, there are three sports the Athlyte Search Dashboard supports:

  • Men's Football
  • Men's Basketball
  • Women's Basketball

Features

Search Type

There are two sorts of stat searches that can be performed on every sport:

  • Player-based searches
  • Team -based searches

Result Screen

The result screen displays the results of your query.

Filters

Use the filters on the left side of the result table to further refine your results.

Filter Function
Opponent Conference Filter results based on the selected opponent's conference.
Opponent Select a specific opponent team to filter the matches.
Results Filter by match result (e.g., Win or Loss).
Game Type Filter based on the type of game (e.g., Regular, Postseason).
Period Type Filter results by the time period (e.g., Regular, Overtime).
Date Specify a custom date range to narrow down the results by match date.
Name Search for a specific player by name.
Position Filter players based on their playing position (e.g., RB, QB, WR).
Classes Filter players by class year (e.g., Freshman, Sophomore, Junior, Senior).

*Note: Highlighted filters apply only to Player-based searches and are not applicable for Team-based searches.

Player Dashboard

To get a detailed view of a player's statistics, including their performance in various games, click on the player name. This will take you to the Player Dashboard.
